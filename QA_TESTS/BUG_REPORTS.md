# TC-02:
* Попытка проанализировать текстовый файл, который не является логом

## Шаги для воспроизведения:
* -f report.txt

## Ожидаемое поведение:
* Программа не падает. Все строки, которые она не смогла распознать
как уровни логов (INFO/ERROR), записываются в категорию UNKNOWN

## Результат
* FAILED (тест не упал, но выдал неверную информацию в подсчете логов)

## Статус
* FIXED (добавлена проверка разделителя "|" перед уровнем лога,
чтобы избежать ложных срабатываний)

# TC-04:

## Шаги для воспроизведения:
* Запустить анализатор с флагом -f и указать путь к любой картинке .jpg

## Ожидаемое поведение:
* Ожидаемый результат: Программа выводит сообщение об ошибке формата
и корректно завершается

## Результат
* FAILED (Программа прерывается системной ошибкой UnicodeDecodeError)

## Статус
* FIXED (добавлена обработка исключений через try-except)
